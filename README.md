## 📱 322web - Портфолио Сайт

Энэ бол Монголын веб хөгжүүлэгч Bat-Ochir-ийн портфолио веб сайт юм. Firebase болон JavaScript ашиглан бүтээгдсэн.

### ✨ Онцлог

- 🎨 **Үзэсгэлэнтэй Design** - Modern UI/UX сайн практик
- 🔐 **Admin Панель** - Контентыг удирдахуйцмөрөгөлтөн үйлдлүүд
- 🔒 **Firebase Authentication** - Аюулгүй admin нэвтрэлт
- 📱 **Responsive Design** - Бүх төхөөрөмжид ажилладаг
- 🚀 **Real-time Updates** - Firebase Firestore ашиглан амжилттай

### 📋 Үндсэн Хэсгүүд

#### 🏠 Нүүр Хэсэг (Homepage)
- Персональ танилцуулга
- Ур чадварын жагсаалт
- Төслүүдийн галерей (зургатай)
- Хичээлийн явц

#### 🛠️ Admin Панель (`/admin`)
- **Нэвтрэлт хэсэг** - Email/Password-аар нэвтрэх
- **Контент Удирдалт**
  - Сайтын тохиргоо засах
  - Ур чадварууд нэмэх/засах/хасах
  - Төслүүд нэмэх/засах/хасах
  - Хичээлүүд нэмэх/засах/хасах

### 🔧 Техникийн Стек

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth
- **Hosting**: Vercel
- **Design**: Responsive Grid Layout
